# i18n — Мультиязычность

## Правила

- Все пользовательские строки ТОЛЬКО через `t('namespace.key')` — без исключений
- При добавлении нового ключа — сразу в `en.json` и `ru.json`
- Namespace = секция или страница: `hero`, `nav`, `footer`
- Язык хранится в `localStorage`

---

## Настройка i18next

```tsx
// src/i18n.ts
import i18n from 'i18next'
import { initReactI18next } from 'react-i18next'
import en from './locales/en.json'
import ru from './locales/ru.json'

i18n.use(initReactI18next).init({
  resources: {
    en: { translation: en },
    ru: { translation: ru },
  },
  lng: localStorage.getItem('lang') ?? 'en',
  fallbackLng: 'en',
  interpolation: { escapeValue: false },
})

export default i18n
```

---

## Использование в компонентах

```tsx
const { t } = useTranslation()

// Базовое
<h1>{t('hero.title')}</h1>

// С интерполяцией
<p>{t('section.deposit', { amount: '$250' })}</p>
// en.json: "deposit": "Minimum Deposit: {{amount}}"

// Множественное число
<p>{t('results.count', { count: 5 })}</p>
// en.json: "count_one": "{{count}} result", "count_other": "{{count}} results"
```

## Хук переключения языка

```tsx
// src/hooks/useLanguage.ts
import { useTranslation } from 'react-i18next'

const SUPPORTED = ['en', 'ru'] as const
type Lang = typeof SUPPORTED[number]

export const useLanguage = () => {
  const { i18n } = useTranslation()

  const change = (lang: Lang) => {
    i18n.changeLanguage(lang)
    localStorage.setItem('lang', lang)
  }

  return {
    current: i18n.language as Lang,
    change,
    languages: SUPPORTED,
  }
}
```

## Структура ключей

```json
{
  "common": {
    "cta": "Get Started",
    "learnMore": "Learn More",
    "loading": "Loading...",
    "error": "Something went wrong"
  },
  "nav": {
    "home": "Home",
    "login": "Log In"
  },
  "hero": {
    "title": "[Заголовок]",
    "subtitle": "[Подзаголовок]"
  },
  "footer": {
    "rights": "© 2025 [Название]. All rights reserved."
  }
}
```

Расширяй под нужды проекта. Namespace — всегда название секции или страницы.
